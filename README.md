# Snake Rust
Snake Game with WASM + Rust

## Installation
1. Clone the repository
```bash
git clone https://github.com/zuramai/snake-rust/
```

2. Compile 
```bash
wasm-pack build --target web 
```

3. Open `index.html` in the browser
